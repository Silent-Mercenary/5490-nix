# 5490-nix

Main personal nix config for Dell Latitude 5490
CPU: i5-8250u
GPU: Intel UHD Graphics 620
RAM: 16GB

Primary use: Personal and School

Main DE: MangoWC + Noctalia Shell
Window Manager: GreetD + regreet
Terminals: GhosTTY and Kitty with fish as primary shell

###Gaming
Includes 
* Steam
* Heroic Games Launcher
* Lutris
Vulkan works OOTB with very limited performance. **For Light Gaming Only** 
(Most you can play with dogwater settings is warframe)

###Communication
Includes
* Signal
* Legcord (Discord)
